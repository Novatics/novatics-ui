# checkbox

## Installation

It is available as an [npm package](https://www.npmjs.com/package/novatics)

**npm:**

```sh
npm install @novatics/checkbox
```

**yarn:**

```sh
yarn add @novatics/checkbox
```

## Usage

```jsx
import { Checkbox } from '@Novatics/checkbox';

<Checkbox color="primary" />;
```
