# CurrencyTextField <img src={`https://img.shields.io/badge/Version-v${pkg.version}-brightgreenn`} />

Use this component for currency inputs

### Installation

First install the latest version of the package from `@novatics-ui/currency-text-field`.

```bash
yarn add @novatics-ui/currency-text-field
```

### Dependencies

This component extends [Material UI React Text Field](https://mui.com/material-ui/react-text-field). Therefore is needed to install Material UI:

```bash
yarn add @mui/material @emotion/react @emotion/styled
```

### Basic Usage

```tsx

import CurrencyTextField from '@novatics-ui/currency-text-field';

<CurrencyTextField
  value={123}
  onChange={() => {
    /* setup on change */
  }}
/>
```
