### Installation

First install the latest version of the package from `@novatics/linear-progress`.

```shell
  yarn add -D @novatics/linear-progress
```

and use it

```tsx
import ... from '@novatics/linear-progress';

  ...


```

### Dependencies

This component extends [Material UI React Linear Progress](https://mui.com/material-ui/react-progress/). Therefore is needed to install Material UI:

```bash
yarn add @mui/material @emotion/react @emotion/styled
```

### Basic Usage

The LinearProgress component is used to show a progress bar. The `withLabel` attribute is used to present a label with the percentage of the progress.
