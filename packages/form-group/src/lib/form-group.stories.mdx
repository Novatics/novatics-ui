### Installation

First install the latest version of the package from `@novatics/form-group`.

```shell
  yarn add -D @novatics/form-group
```

### Basic Usage

The `FormGroup` component groups input components together. It can be used to create sections in a form.

<Canvas>
  <Story name="Basic Usage">
    {(args) => {
      return (
        <FormGroup title={'Project Settings'} direction="column">
          <InputField
            label="Project's name"
            required
            hint="Some important text"
          />
          <InputField label="Key" required />
        </FormGroup>
      );
    }}
  </Story>
</Canvas>

### Spacing

The `titleSpacing` prop can be used to regulate the spacing between the title and the children, and the `childrenSpacing` component can be used to control the spacing between the children.

<Canvas>
  <Story name="Spacing">
    {(args) => {
      return (
        <FormGroup
          title={'Smaller Title Spacing'}
          titleSpacing={1}
          childrenSpacing={3.5}
          direction="column"
        >
          <InputField
            label="First child"
            required
            hint="Bigger spacing between children"
          />
          <InputField label="Second child" required />
        </FormGroup>
      );
    }}
  </Story>
</Canvas>

### Direction

The `direction` prop can be used to change the direction of the children. The default value is `column`.

<Canvas>
  <Story name="Direction">
    {(args) => {
      return (
        <FormGroup title={'Project Settings'} direction="row">
          <InputField label="First" required hint="Some important text" />
          <InputField label="Second" required />
          <InputField label="Third" required />
        </FormGroup>
      );
    }}
  </Story>
</Canvas>
