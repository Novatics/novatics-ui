### Installation

First install the latest version of the package from `@novatics/link`.

```shell
  yarn add -D @novatics/link
```

```shell
  npm install --dev @novatics/link
```

and use it

```tsx
import ... from '@novatics/link';

  ...

```

### Basic Usage

The Link component is used to display a customizable anchor element. The isExternal attribute is
used to show the external icon. The Link component accepts any Typography props passed to it.
