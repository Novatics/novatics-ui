# FromUrl <img src={`https://img.shields.io/badge/Version-v${pkg.version}-brightgreenn`} />

### Installation

First install the latest version of the package from `@novatics/file-upload`.

```bash
yarn add @novatics/file-upload
```

### Basic Usage

The FromUrl component is part of the FileUpload component. It can be used to upload files from an URL. It has two variants: `single` and `combined`.

<Canvas>
  <Story name="Basic Usage">
    {(args) => {
      return (
        <Stack spacing={2}>
          <FromUrl variant="single" />
          <FromUrl acceptLabels=".CSV, .XLSX" variant="combined" />
        </Stack>
      );
    }}
  </Story>
</Canvas>

### States

In addition to the default state, the FromUrl component has a disabled and an error state, defined by the `disabled` and `error` props respectively.

<Canvas>
  <Story name="States">
    {(args) => {
      return (

          <FromUrl variant="combined" error="An error occurred." />
        </Stack>
      );
    }}
  </Story>
</Canvas>
