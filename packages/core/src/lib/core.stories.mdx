### Basic Usage

First install the latest version of the package from `@novatics-ui/core`.


```shell
  yarn add -D @novatics-ui/core
```

```shell
  npm install --dev @novatics-ui/core
```


and use it

```tsx
import { Button, CurrencyTextField, ...} from '@novatics-ui/core';
```

>  All components are exported by the core
