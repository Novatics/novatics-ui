### Installation

First install the latest version of the package from `@novatics/alert`.

```shell
  yarn add -D @novatics/alert
```

```shell
  npm install --dev @novatics/alert
```

and use it

```tsx
import ... from '@novatics/alert';

  ...


```

### Basic Usage

The `Alert` component have three variants: it supports outlined, contained and only text styling. The default variant is `text`.
