### Basic Usage

First install the latest version of the package from `@novatics/core`.

```shell
  yarn add -D @novatics/core
```

```shell
  npm install --dev @novatics/core
```

and use it

```tsx
import { Button, CurrencyTextField, ...} from '@novatics/core';
```

> All components are exported by the core
