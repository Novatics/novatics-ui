# Breakpoints <img src={`https://img.shields.io/badge/Version-v${pkg.version}-brightgreenn`} />

This API enables the use of breakpoints in a wide variety of contexts.

For optimal user experience, novatics ui uses material ui as a base to adapt to various layouts.

## Default Breakpoints

Each breakpoint [key] matches with a fixed screen width [value]:

- **xxs**: extra-extra-small: 0px
- **xs**: extra-small: 600px
- **sm**: small: 960px
- **md**: medium: 1024px
- **lg**: large: 1200px
- **xl**: extra-large: 1440px
- **xxl**: extra-extra-large: 1728px
