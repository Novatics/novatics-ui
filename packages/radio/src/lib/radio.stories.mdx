### Installation

First install the latest version of the package from `@novatics/radio`.

```shell
  yarn add -D @novatics/radio
```


### Dependencies

This component extends [Material UI React Radio](https://mui.com/material-ui/react-radio-button). Therefore is needed to install Material UI:

```bash
yarn add @mui/material
```
