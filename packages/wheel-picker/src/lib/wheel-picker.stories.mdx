### Installation

First install the latest version of the package from `@novatics/wheel-picker`.

```shell
  yarn add -D @novatics/wheel-picker
```

```shell
  npm install --dev @novatics/wheel-picker
```

and use it

```tsx
import ... from '@novatics/wheel-picker';

  ...


```
