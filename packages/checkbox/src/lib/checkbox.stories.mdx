### Installation

First install the latest version of the package from `@novatics-ui/checkbox`.

```shell
  yarn add -D @novatics-ui/checkbox
```

### Dependencies

This component extends [Material UI React Checkbox](https://mui.com/material-ui/react-checkbox/). Therefore is needed to install Material UI:

```bash
yarn add @mui/material
```
