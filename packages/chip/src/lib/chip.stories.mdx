# Chip <img src={`https://img.shields.io/badge/Version-v${pkg.version}-brightgreenn`} />

### Installation

First install the latest version of the package from `@novatics/chip`.

```shell
  yarn add -D @novatics/chip
```

### Basic Usage

The `Chip` component have two variants: it supports outlined and filled styling. The default variant is filled.

<Canvas>
  <Story name="Basic Usage">
    {(args) => {
      return (
        <div
          style={{
            display: 'flex',

            alignItems: 'center',
            gap: '10px',
          }}
        >
          <Chip label="Chip" />
          <Chip label="Chip" variant="outlined" />
        </div>
      );
    }}
  </Story>
</Canvas>

### Click interaction

Chips can be clickable. In this case, it will have a hover and a click effect. It can be set by either passing the `clickable` prop or by passing a function to `onClick` prop.

<Canvas>
  <Story name="Click interaction">
    {(args) => {
      return (
        <div
          style={{
            display: 'flex',
            flexDirection: 'row',
            alignItems: 'center',
            gap: '10px',
          }}
        >
          <Chip label="Clickable Chip" clickable />
          <Chip
            label="Click Me"
            onClick={() => window.alert('Hi! :)')}
            variant="outlined"
          />
        </div>
      );
    }}
  </Story>
</Canvas>

### Delete interaction

A `Chip` can present a delete icon by passing a `onDelete` prop.

<Canvas>
  <Story name="Delete interaction">
    {(args) => {
      return (
        <div

            display: 'flex',
            flexDirection: 'row',
            alignItems: 'center',
            gap: '10px',
          }}
        >
          <Chip label="Deletable Chip" onDelete={() => {}} />
          <Chip label="Deletable Chip" onDelete={() => {}} variant="outlined" />
          <Chip
            label="Deletable and Clickable"
            onDelete={() => {}}
            clickable
            variant="outlined"
          />
        </div>
      );
    }}
  </Story>
</Canvas>
