# Divider <img src={`https://img.shields.io/badge/Version-v${pkg.version}-brightgreenn`} />

### Installation

First install the latest version of the package from `@novatics/divider`.

```shell
  yarn add -D @novatics/divider
```

### Dependencies

This component extends [Material UI React Divider](https://mui.com/material-ui/react-divider/). Therefore is needed to install Material UI:

```bash
yarn add @mui/material
```
