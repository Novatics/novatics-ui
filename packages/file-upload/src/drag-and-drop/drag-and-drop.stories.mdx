### Installation

First install the latest version of the package from `@novatics/file-upload`.

```bash
yarn add @novatics/file-upload
```

### Basic Usage

The DragAndDrop component is part of the FileUpload component. It can be used to upload files by dragging and drop files or by clicking on the button to select them. It has two variants: `fullscreen` and `inline`.

### States

In addition to the default state, the DragAndDop component has a disabled and an error state, defined by the `disabled` and `error` props respectively.
