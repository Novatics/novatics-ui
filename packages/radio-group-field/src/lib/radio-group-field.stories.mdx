### Installation

First install the latest version of the package from `@novatics-ui/radio-group-field`.

```shell
  yarn add -D @novatics-ui/radio-group-field
```

### Basic Usage

The `RadioGroupField` component allows the creation of a field composed of radio buttons.

<Canvas>
  <Story name="Basic Usage">
    {(args) => {
      return (
        <RadioGroupField
          label="Is this a radio group?"
          withTextHelper={false}
          options={[
            {
              label: 'Yes',
              value: 'yes',
            },
            {
              label: 'Probably',
              value: 'probably',
            },
            {
              label: 'No',
              value: 'no',
              disabled: true,
            },
          ]}
          {...args}
        />
      );
    }}
  </Story>
</Canvas>

### Direction

The `direction` prop allows the user to change the direction of the radio buttons. The default value is `column`

<Canvas>
  <Story name="Direction">

</Canvas>

### Helper Text

The `withTextHelper` prop allows the user to add a hint or show an error to the field. By default, it is set to `true`.
