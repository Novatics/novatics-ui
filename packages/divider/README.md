# divider

## Installation

It is available as an [npm package](https://www.npmjs.com/package/novatics)

**npm:**

```sh
npm install @novatics/divider
```

**yarn:**

```sh
yarn add @novatics/divider
```

## Usage

```jsx
import Divider from '@novatics/divider';

<div>
  <Divider {...args} />
</div>;
```
