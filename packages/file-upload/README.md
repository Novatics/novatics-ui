# file-upload

## Installation

It is available as an [npm package](https://www.npmjs.com/package/novatics)

**npm:**

```sh
npm install @novatics/file-upload
```

**yarn:**

```sh
yarn add @novatics/file-upload
```

## Usage

```jsx
import FileUpload from '@novatics/file-upload';

<FileUpload />;
```
