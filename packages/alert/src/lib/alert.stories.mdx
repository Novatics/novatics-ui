### Installation

First install the latest version of the package from `@novatics/alert`.

```shell
  yarn add -D @novatics/alert
```

```shell
  npm install --dev @novatics/alert
```

and use it

```tsx
import ... from '@novatics/alert';

  ...


```

### Basic Usage

The `Alert` component supports three type variants: persistent, dispensable and action alerts.

<Canvas>
  <Story name="Basic Usage">

              <Stack spacing={2}>
                <Typography variant="body">Persistent</Typography>
                <Alert severity="info">This is a persistent alert.</Alert>
              </Stack>
            </Grid>
            <Grid item xxs={4}>
              <Stack spacing={2}>
                <Typography variant="body">Dispensable</Typography>
                <Alert onClose={() => undefined}>
                  This is a dispensable alert.
                </Alert>
              </Stack>
            </Grid>
            <Grid item xxs={4}>
              <Stack spacing={2}>

                  This is an alert with some action!
