### Installation

First install the latest version of the package from `@novatics/checkbox`.

```shell
  yarn add -D @novatics/checkbox
```


### Dependencies

This component extends [Material UI React Checkbox](https://mui.com/material-ui/react-checkbox-button). Therefore is needed to install Material UI:

```bash
yarn add @mui/material
```
