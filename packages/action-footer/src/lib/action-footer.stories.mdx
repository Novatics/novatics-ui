### Installation

First install the latest version of the package from `@novatics/action-footer`.

```shell
  yarn add -D @novatics/action-footer
```

### Dependencies

This component extends [Material UI React Stack](https://mui.com/material-ui/react-stack). Therefore is needed to install Material UI:

```bash
yarn add @mui/material
```
