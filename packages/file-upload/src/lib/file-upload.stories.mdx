# FileUpload <img src={`https://img.shields.io/badge/Version-v${pkg.version}-brightgreenn`} />

### Installation

First install the latest version of the package from `@novatics/file-upload`.

```bash
yarn add @novatics/file-upload
```

### Dependencies

This component extends [Material UI React](https://mui.com/material-ui/) components. Therefore is needed to install Material UI:

```bash
yarn add @mui/material @emotion/react @emotion/styled
```

### Basic Usage

The FileUpload is composed by 3 components: `FromUrl`, `DragAndDrop` and `ListItem`. They can be used separately or combined.

<Canvas>
  <Story name="Basic Usage">
    {(args) => {
      return (
        <Stack width={500} spacing={2}>
          <DragAndDrop
            variant="inline"
            multiple
            acceptedLabels="JPG, PNG and GIF"
          />
          <Divider>or</Divider>
          <FromUrl
            accept="image/jpeg"
            acceptLabels=".JPEG"
            variant="combined"
          />
          <br />
          <ListItem fileName="Test.png" loadingPercentage={25} />
        </Stack>
      );
    }}
  </Story>
</Canvas>

### From URL

The FromUrl component receives an URL and a callback function to be called when the user clicks on the button to upload the file. It has two variants: `single` and `combined`.

<Canvas>
  <Story name="From URL">
    {(args) => {
      return (
        <Stack spacing={2}>
          <FromUrl variant="single" />
          <FromUrl acceptLabels=".CSV, .XLSX" variant="combined" />
        </Stack>
      );
    }}
  </Story>
</Canvas>

### Drag and Drop

The `DragAndDrop` component can be used in two variants: `inline` and `fullscreen`. Both draggin and dropping files or clicking on the button to select them are supported.

<Canvas>
  <Story name="Drag and Drop">
    {(args) => {
      return (
        <Stack spacing={2}>
          <DragAndDrop
            variant="fullscreen"
            multiple
            acceptedLabels="JPG, PNG and GIF"
          />
          <DragAndDrop variant="inline" acceptedLabels="JPG, PNG and GIF" />
        </Stack>
      );
    }}
  </Story>
</Canvas>

### List Item

A list item can be used to show the status of a file upload. If it is loading, succeeded or failed.

<Canvas>
  <Story name="List Item">
    {(args) => {
      return (
        <Stack width="50rem">
          <ListItem fileName="Test.jpg" loadingPercentage={25} />
          <br />
          <ListItem
            fileName="Test.csv"
            succeeded
            onSuccessClick={() => {}}
            deleteAfterSuccess
          />
          <br />
          <ListItem fileName="Test.txt" error="Tente novamente" />
        </Stack>
      );
    }}
  </Story>
</Canvas>
